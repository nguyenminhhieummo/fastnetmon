Class XML

Description
Description | Vars (details) | Methods (details)

An

  1. XML
object represents XML content and provides methods to manipulate it.

The

  1. XML
object has an 'active' element, which represents the hierarchy at which the XML can be manipulated. <p> As an example, one
  1. creates a XMLBuilder object.
  2. create a configuration as an XML object.
  3. Call the loadXMLConfiguration(XML) method on Device

		append (line 133)
	
 
	
	

Append an element under the active elment of XML object.


This function takes variable no of arguments. 
  1. only element: Append an element under the active element
of XML object. The new element now becomes the active element.
  2. one element and one text : Append an element, with text, under the
active element of XML object. The new element now becomes
the active element.
  3. one element and an arrya of text values : Append multiple elements,
with same name but differrent text under the active element of XML object.
  4. An associative array : Append multiple elements with different names
and different text, under the active element of XML object.In this,
each entry contains element name as the key and text value as the
key value.
  5. An element and an associative array : Append an element under the
active element of XML object. The new element now becomes the
active element.Then,append multiple elements wiht different names
and different text, under the new active element.
